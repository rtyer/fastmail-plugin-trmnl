# Contributing

Thanks for helping improve this project.

## Development

```sh
npm install
npm run typecheck
npm test
npm run build
```

## Calendar Fixtures

Keep test fixtures synthetic. Do not commit exported personal calendars, real published ICS URLs, or screenshots containing real events.

## Pull Requests

- Keep changes focused.
- Add or update tests for calendar normalization behavior.
- Run `npm run typecheck`, `npm test`, and `npm run build` before submitting.
